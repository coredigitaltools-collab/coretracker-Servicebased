# Service Business Tracker
### by CoreDigitalTools

A PWA (Progressive Web App) for service business owners. Track clients, appointments, income, expenses and your pricing list — with owner and staff access levels.

---

## How to deploy on GitHub Pages

1. Create a new repository on GitHub (e.g. `service-business-tracker`)
2. Upload all files keeping this exact folder structure:
   ```
   index.html
   manifest.json
   sw.js
   icons/
     icon-72x72.png
     icon-96x96.png
     icon-128x128.png
     icon-144x144.png
     icon-152x152.png
     icon-192x192.png
     icon-384x384.png
     icon-512x512.png
   ```
3. Go to your repository **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)`
5. Click **Save**
6. Your app will be live at: `https://yourusername.github.io/service-business-tracker`

## How clients install it as an app

**On desktop (Chrome):**
- Visit the link
- Click the install icon in the browser address bar, or click "Install" on the banner
- App installs to their desktop like a normal app

**On Android:**
- Visit the link in Chrome
- Tap "Add to Home Screen" when prompted
- App installs with the CoreDigitalTools icon

**On iPhone/iPad:**
- Visit the link in Safari
- Tap the Share button → "Add to Home Screen"
- App installs to the home screen

## Features

- **Owner login** — PIN protected, full access to all tabs
- **Staff login** — no PIN needed, access to clients and appointments only
- **Dashboard** — daily, weekly and monthly revenue charts + summaries
- **Clients** — add and manage clients with contact info
- **Appointments** — log bookings linked to clients and services
- **Income** — track payments, paid vs unpaid, daily/monthly/annual totals
- **Expenses** — log costs by category, see net profit
- **Pricing list** — add services with optional duration and category
- **Works offline** — data saved locally, no internet needed after first load
- **Dark mode** — automatically follows device theme
- **Multi-currency** — UGX, KES, TZS, RWF, USD, GBP, EUR, NGN, ZAR, GHS
